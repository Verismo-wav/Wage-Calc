# Wage Calculator App

A comprehensive financial planning tool that visualizes how your wages are allocated across expenses, taxes, and savings.

## Features
- Wage calculations across different time periods
- Tax calculations
- Monthly expense tracking with saved profiles
- Loan cost analysis with additional principal payments
- Visual time and work allocation breakdowns
- Retirement savings projections

## Deployment to Vercel

### Step 1: Upload to GitHub
1. Create a new repository on GitHub
2. Upload all the files from this project to your repository

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up (use your GitHub account)
2. Click "Add New Project"
3. Select your GitHub repository
4. Vercel will auto-detect the settings
5. Click "Deploy"
6. Your app will be live in 1-2 minutes!
